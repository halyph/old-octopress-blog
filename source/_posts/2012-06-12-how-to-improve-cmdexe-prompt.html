--- layout: post title: How to improve cmd.exe prompt date: '2012-06-12T20:38:00.001+03:00' author: Orest Ivasiv categories: - cmd.exe - console modified_time: '2012-06-12T20:43:08.700+03:00' thumbnail: http://2.bp.blogspot.com/-oZ79-qCc15o/T9d6epWhOSI/AAAAAAAAAnU/qnahfAD-xlw/s72-c/Clipboard-1.png blogger_id: tag:blogger.com,1999:blog-7372777165432727866.post-7989723756897266082 blogger_orig_url: http://www.halyph.com/2012/06/how-to-improve-cmdexe-prompt.html --- I'm sure that you have some trouble with "too long command prompt in windows cmd.exe" and it is annoying:

Fortunately, it can be easily fixed with PROMPT command (see official help): 

> prompt /?

PROMPT [text]

  text    Specifies a new command prompt.

Prompt can be made up of normal characters 
and the following special codes:

  $A   & (Ampersand)
  $B   | (pipe)
  $C   ( (Left parenthesis)
  $D   Current date
  $E   Escape code (ASCII code 27)
  $F   ) (Right parenthesis)
  $G   > (greater-than sign)
  $H   Backspace (erases previous character)
  $L   < (less-than sign)
  $N   Current drive
  $P   Current drive and path
  $Q   = (equal sign)
  $S     (space)
  $T   Current time
  $V   Windows version number
  $_   Carriage return and linefeed
  $$   $ (dollar sign)

I've configured PROMPT User variable to apply my custom PROMPT setting: [$P]$_$$$S
